Quick guide

  • Green dot — the device is online right now. A grey dot means it’s offline (you’ll see when it was last seen).
  • exit node An exit node routes a router’s internet traffic through it. Only approved exit nodes can be selected.
  • router A router you control. Pick where its internet goes with “Route internet through”.
  • ↑↓ Stats show traffic carried through the current exit node, and when it last completed a handshake.
